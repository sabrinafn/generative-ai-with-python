1. O passo a passo para obter uma chave de api.

    - On Google AI Studio, clicked on 'Create API Key'.
        - Name the key and create/select a project to add it to.
        - After creation, you can copy the key.
    - Set the key as an environment variable:
        - Check which shell it's being used with: `echo $SHELL`
        - Open to edit the configuration file with:
            - if bash: `vim ~/.bashrc`
            - if zsh: `vim ~/.zshrc`
        - Add this line to the file: `export GEMINI_API_KEY=<API_KEY_HERE>`
        - Apply changes by running:
            - if bash: `source ~/.bashrc`
            - if zsh: `source ~/.zshrc`
    
    {https://ai.google.dev/gemini-api/docs/api-key}
    {https://ai.google.dev/gemini-api/docs/api-key#set-api-env-var}
